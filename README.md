## 🧬 GEO-DAM Framework Nedir?

**GEO-DAM** (*Generative Engine Optimization - Dijital Algı Mimarisi*), yapay zeka motorlarında (ChatGPT, Google AI Overviews, Perplexity vb.) markaların dijital algısını baştan inşa etmek için Yusuf Şahin tarafından geliştirilmiş açık kaynak bir metodoloji çerçevesidir.

- **Framework Deposu:** [github.com/yusufads/GEO-DAM](https://github.com/yusufads/GEO-DAM/tree/main/framework)
- **Akademik Kayıt:** [Zenodo / CERN](https://zenodo.org/records/20069077)
- **Lisans:** MIT
